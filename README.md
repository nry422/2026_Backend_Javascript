# Backend JavaScript — IFSC Jaraguá do Sul RAU

Exercícios de JavaScript desenvolvidos em aula durante o segundo semestre do curso de desenvolvimento de sistemas no IFSC Jaraguá do Sul RAU.

---

Aulas

20/02/2026 — Introdução ao JavaScript
Primeiros contatos com JavaScript no navegador.

- Uso de `alert()` e `console.log()`
- Variáveis com `const` e `let`
- Template literals com `${}`
- Script inline vs arquivo externo (`.js`)
- Manipulação básica do DOM com `querySelector`
- Captura de eventos de formulário com `addEventListener`

---

27/02/2026 — Formulários e Cálculos
Integração entre HTML e JavaScript para processar entradas do usuário.

- Captura de múltiplos campos de formulário
- Conversão de tipos com `Number()`
- Cálculo e formatação de resultados
- Uso de `Math.floor()` e operador `%` para converter minutos em horas
- Exibição de resultados em elementos `h3` e `h4`

**Exemplos:** conversor de duração de filmes, formulário de nome/idade/cidade.

---

06/03/2026 — Condicionais
Uso de estruturas condicionais para tomada de decisão.

- `if`, `else if`, `else`
- `switch/case` com múltiplos casos e `default`
- Estilização dinâmica via `element.style.color`

**Exemplos:** situação do aluno (aprovado/exame/reprovado), taxa de entrega por bairro, promoção de supermercado, parcelamento de veículos.

---

13/03/2026 — Laços de Repetição
Estruturas de repetição para processar dados múltiplas vezes.

- `while` e `do/while`
- `for` clássico com incremento e decremento
- Validação de entrada com `isNaN()`
- Concatenação de strings em loop

**Exemplos:** tabuada, contagem regressiva, listagem de números pares, repetidor de frutas, verificador de velocidade, parquímetro com troco, classificador de triângulos.

---

20/03/2026 — Vetores (Arrays) e localStorage
Manipulação de arrays e persistência de dados no navegador.

- Declaração e acesso a arrays
- Métodos: `push()`, `pop()`, `shift()`, `unshift()`, `sort()`, `join()`, `split()`
- Iteração com `for` e `forEach()`
- Persistência com `localStorage` (`setItem`, `getItem`, `removeItem`, `clear`)

**Exemplos:** fila de pacientes do dentista (com urgência e atendimento), lista de compras ordenada com persistência, contador com incremento por F5.

---

Cada pasta contém arquivos `.html` que podem ser abertos diretamente no navegador. Não é necessário servidor ou instalação adicional, porém recomendo XAMPP com servidor Apache :)
